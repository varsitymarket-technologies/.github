# Varsitymarket Technologies

> **Empowering businesses with lightweight, integrated e-commerce and web solutions**

Welcome to Varsitymarket Technologies, a South African tech startup based in Johannesburg building innovative solutions that help businesses establish and scale their digital presence.

---

## 🚀 About Us

Varsitymarket Technologies is a forward-thinking tech startup specializing in creating **lightweight, embedded solutions** that seamlessly integrate with existing web platforms. Our mission is to democratize e-commerce technology by providing affordable, easy-to-implement solutions for businesses of all sizes.

Based in South Africa, we're committed to delivering quality software that makes a real impact in the digital marketplace.

**Website:** [varsitymarket.co.za](https://varsitymarket.co.za/)

---

## 🎯 What We Do

We develop:

- **Embedded E-Commerce Microservices** – Lightweight plugins that turn static websites into fully-featured online stores
- **Cloud Management Solutions** – Serverless project management for static websites
- **Web Infrastructure** – Custom themes and tools to power modern web experiences
- **Development Tools** – Building blocks for developers who want flexible, modular solutions

---

## 📦 Our Key Projects

### 1. **vm.embedded.store** 
A lightweight, embedded e-commerce microservice designed as a plugin for static websites.

- 📍 Repository: [vm.embedded.store](https://github.com/varsitymarket-technologies/vm.embedded.store)
- 🛠 **Tech Stack:** PHP
- ✨ **Features:**
  - Customer-facing shop interface
  - Comprehensive admin control panel
  - Easy integration with static sites
  - Complete e-commerce functionality in a compact package

### 2. **vmTECH-cloud-manager**
Serverless static website project manager for seamless deployment and management.

- 📍 Repository: [vmTECH-cloud-manager](https://github.com/varsitymarket-technologies/vmTECH-cloud-manager)
- 🛠 **Tech Stack:** PHP
- 📜 **License:** GPL-3.0
- ✨ **Purpose:** Simplify management of static websites without complex infrastructure

### 3. **embedded-themes**
A collection of customizable themes designed for our embedded systems.

- 📍 Repository: [embedded-themes](https://github.com/varsitymarket-technologies/embedded-themes)
- 🛠 **Tech Stack:** JavaScript
- ✨ **Purpose:** Provide flexible, modern theming options for embedded stores

### 4. **vmbuilder**
The builder component powering Varsitymarket's development toolkit.

- 📍 Repository: [vmbuilder](https://github.com/varsitymarket-technologies/vmbuilder)
- 🛠 **Tech Stack:** PHP

### 5. **tech-space**
Our main technical website and documentation hub.

- 📍 Repository: [tech-space](https://github.com/varsitymarket-technologies/tech-space)
- 🛠 **Tech Stack:** HTML

### 6. **about-us**
A quick introduction to our mission and team.

- 📍 Repository: [about-us](https://github.com/varsitymarket-technologies/about-us)
